# [Monitev(https://monitev.com) [V1.1]]

![alt text](https://i.postimg.cc/T34qwFLS/monitev.png? "Monitev")


Monitev es una plataforma para el gestión, captura, procesamiento, almacenamiento, analisis y visualización de datos provenientes de dispositivos IOT.
Cuenta con herramientas de gran alta calidad para la manipulación de datos, además con una variedad de Widget que se pueden personalizar en la interface de usuario
Monitev lleva en el mercado desde el 2019, con sede en la ciudad de Cali.
